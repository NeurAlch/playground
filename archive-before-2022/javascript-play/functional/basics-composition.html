Associative

Composition of functions is always associative.
The identity function is both a left and a right identity for composition.
Function composition can be used in correspondence with function iteration. 

E.strictEqual(
    F.compose(F.compose(inc, inc), inc)(1),
    F.compose(inc, F.compose(inc, inc))(1)
);
